# Website.

This is a simple SvelteKit template that reads data from YAML files and creates resume and portfolio pages

## Features

- Minimal
- Fast
- Responsive
- Printer friendly
- ATS friendly
- 100 Lighthouse score
- No JS

## How to use

```
npm install
npm run dev
```

Edit Yaml files on `src/data` and put your own data
